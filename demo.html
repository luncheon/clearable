input clear icon

Show a clear icon on the focused <input>.

Usage

  1. Apply a stylesheet. <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/input-clear-icon/input-clear-icon.min.css">
  2. Load a script. <script src="https://cdn.jsdelivr.net/npm/input-clear-icon/input-clear-icon.min.js"></script>
  3. Place input elements as usual.

Styling

  • "input-clear-icon" is the tag name of the clear icon that can be used as a CSS selector.
  • "data-input-clear-icon-class" attribute of the input element is applied as the class name.
  • "data-input-clear-icon-style" attribute of the input element is applied as the inline style.
<input value="default">
<style>
  .my-clear-icon { color: rgb(64, 128, 255); }
  .my-clear-icon:hover { color: white; background-color: rgb(64, 128, 255); }
</style>
<input
  data-input-clear-icon-class="my-clear-icon"
  value="blue"
>
<input
  data-input-clear-icon-style="color: #ff4040; border-color: #ff8080; background-color: #fff0f0; border-radius: 50%; font-size: 9px;"
  value="red-circle"
>
